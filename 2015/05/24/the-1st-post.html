octopress3.0 rifyll-theme

Hi people, I choose Octopress 3 and theme it with rifyll as my personal cat-style blog. This blogging framework for hacker cool known for programmers blog about their ideas. Octopress features - (more)

  • Pygments syntax highlighting & Markdown based posts
  • Out of the box support for publishing straight to GitHub pages
  • Built in support for third-party services like Google Analytics

Installation steps:

step 1 Install Jekyll and Octopress 3.

1 # Installing Octopress 3 automatically installs Jekyll as a dependency.
2 $ gem install octopress

step 2 Download Rifyll themed Octopress blog scaffold.

step 3 Cd to Rifyll directory & New a post

1 # This generates a new post yyyy-mm-dd-title.markdown under `_post`.
2 $ cd rifyll-master
3 $ octopress new post "title"

step 4 Build _site

1 # This builds a blog site dir under `_site`.
2 $ jekyll build

optional step 5 Preview locally (NOTE before deployment please kill this process)

1 # This runs a server up at port 4000.
2 # Server address: http://127.0.0.1:4000/
3 $ jekyll serve &

step 6 Set up deployment of website to GitHub

1 # setup method of deployment to github
2 $ octopress deploy init git --url git@github.com:NAME/NAME.github.io.git

step 7 Deploy the built website

1 # deploy built site
2 $ octopress deploy
